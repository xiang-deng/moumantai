# Code of Conduct

Be respectful, constructive, and kind. Harassment, personal attacks, and other
unprofessional conduct are not welcome here.

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)
v2.1. To report an issue, open one at
https://github.com/xiang-deng/moumantai/issues (or use GitHub's private report
feature for sensitive matters). Maintainers may remove contributions or
participants that violate these norms.
